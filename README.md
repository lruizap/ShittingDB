# ShittingDB

Welcome to ShittingDB, a project that aims to provide a fun and easy to use platform to enjoy and share jokes with others.

## Aim

The aim of this project is to create a web and mobile application that allows users to view, share and vote on jokes, as well as upload their own jokes after a moderator review.

### Features

* Attractive and easy to use user interface.

* Functionalities to view, share and vote jokes

* Database for storing jokes

* Compatibility with mobile and web devices

* Search and sort functions for jokes

* Authentication to avoid uploading jokes

* Authentication to prevent uploading of repeated jokes

* Moderation of submissions

* Moderation of jokes uploaded by users

### Technologies

* Backend: Django

* Frontend: Next.js

### Project structure

#### The project is divided into the following folders and files

* backend (Django project)

* frontend (Next.js project)

* db.sqlite3 (SQLite database for development)

* README.md (project information)
